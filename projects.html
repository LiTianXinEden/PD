Website for a Bakery App
Category:

Development

Language & Tools Used:

NodeJs, Express.js, HTML, CSS, phpMyAdmin

Overview:

This is a website made with NodeJs and Express.js along with HTML and CSS. It featured a login page where you can login as either a customer or an admin and the website layout would change according to what you login as. The website makes use of a database, phpMyAdmin, to store the data so the data is not hardcoded and can be updated dynamically

Customers will be able to see a navbar that says "Home, Cart and Logout". Customers are able to add products to their cart and from the cart, they can add more or remove the product from their cart

Unlike Customers, Admins will be able to see "Home, Cart, AdminView, Add Product and Logout" in their navbar. Admins will be able to add new products as well as edit any old products such as changing its image, price or removing the product as a whole. This will update the database that holds the data for this website

CRUD Functionality in a RESTful Supermarket Website
Category:

Development

Language & Tools Used:

NodeJs, Express.js, HTML, CSS, phpMyAdmin

Overview:

This is a mini-supermarket website that makes use of a phpMyAdmin database to manage a dynamic product catalog, implementing full CRUD functionality in a RESTful environment.

Users can perform the following actions:

  • Create: Add new products to the catalog with details such as name, image, quantity, and price.
  • Read: View the current product catalog, exploring available products with all relevant information.
  • Update: Modify existing product details, including image, name, quantity, and price, keeping the catalog up-to-date.
  • Delete: Remove products from the catalog, instantly reflecting changes in the database.

Development of a VR Escape Room Game in Unity
Category:

Development

Language & Tools Used:

Unity, Unity #C

Overview:

One of my development related projects is to create a VR game in Unity and code it using Unity's C#.

The game is a VR Escape Room game set in the garage of an abandoned house. The game's objective is for the player to escape by solving puzzles and fixing a Circuit Box that powers the garage door.

I design and coded 4 sequential puzzles for the game that relied on the completion of the previous puzzle before the player can move on to the next one to help create a sense of progression. I created a GDD (Game Design Document) that kept me focused on the game's theme, narrative, and puzzle structure, ensuring that each element aligned with the overall experience I wanted to create.

Travel List App
Category:

Development

Language & Tools Used:

React, CSS, HTML, Javascript

Overview:

This is a travel list app that I created using React, HTML, CSS and JavaScript.

It features an input bar where users can type in the name of the item they want to pack as well as select the quantity of the item they want to pack.

Afterwards they can press the "Add" button to add their item to the list. Users are able to Sort their item list by Input Order, Alphabetical or Packed Status. Users are also able to tick a checkmark when they are done packing the item or delete an item.

There is a button that allows users to remove and clear all the items in the list. There is also a message that automatically updates to show the user how much they have packed
